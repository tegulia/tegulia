Ggplot2散点图

下面我们将介绍使用ggplot2画散点图以及它的衍生图–泡泡图，我们将发现这两个图的画法在ggplot2中几乎是一模一样的。

基本散点图

library(ggplot2)

#为了保持一致的结果，设置一个seed
set.seed(1234567)
#数据集的创建
df.test_data <- data.frame(x_var = 1:50 + rnorm(50,sd=15),
                           y_var = 1:50 + rnorm(50,sd=2),
                           area=    runif(50,2,15)
                           )
                          
                          

# 画图
ggplot(data=df.test_data, aes(x=x_var, y=y_var)) +
  geom_point()

ggplot语法简介

在这个代码中，ggplot()函数表示要调用ggplot图形系统，其中data参数指定数据来源，aes()指定x轴与y轴的数据变量。geom_point()表示我们要画一个点图。

气泡图

library(ggplot2)

#为了保持一致的结果，设置一个seed
set.seed(1234567)
#数据集的创建
df.test_data <- data.frame(x_var = 1:50 + rnorm(50,sd=15),
                           y_var = 1:50 + rnorm(50,sd=2),area=runif(50,2,15))                          
                          

# 画图,注意我们增加了aes(size=area)
ggplot(data=df.test_data, aes(x=x_var, y=y_var)) +
  geom_point(aes(size=area))

是不是非常简单？

气泡图代码解释

我们看到，两者代码几乎一模一样，只是在geom_point()中，增加了aes(size=area)一个参数，表示该数值映射到数据点的面积上。

这是ggplot的强大之处—-调整几句代码就可以改变整个图表的类型，一旦你了解了ggplot的机制，就可以很快的作出非常漂亮，有洞见和价值的可视化图形。
